# **App Name**: Cloudfolio

## Core Features:

- Hero Section: Visually appealing hero section with a brief introduction and a professional headshot.
- About Me Section: Detailed information about your background, experience, and cloud expertise.
- Skills Showcase: Interactive display of your cloud-focused skills (AWS, Azure, GCP, etc.) with proficiency levels.
- Project Showcase: Unique project presentation with dedicated project pages/sub-websites for each project. These can be either hosted separately or as routes. Provide dummy content which is easily updateable.
- Interactive Cloud Architecture Map: Dynamic cloud architecture diagram (VPC, EC2, S3, etc.). Clicking on a component opens its project page. Uses LLM tool to dynamically determine connections based on keywords in project descriptions.
- CV Download: Downloadable CV (Sanved_Resume.pdf) with proper handling in both light and dark modes.
- Contact Section: Contact form and links to professional profiles (LinkedIn, GitHub, etc.).

## Style Guidelines:

- Dark color scheme for a techy and professional aesthetic, but with a smooth light mode toggle.
- Primary color: Subtle, professional blue (#29ABE2) to evoke a sense of trust and technology.
- Background color: Very dark grey (#222222), nearly black. For dark mode. For light mode, background will be near white (#F0F0F0)
- Accent color: Soft, muted green (#8FBC8F) to complement the primary blue and provide highlights.
- Font pairing: 'Space Grotesk' (sans-serif) for headers paired with 'Inter' (sans-serif) for body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use modern, flat icons related to cloud services and technologies.
- Clean, minimalist layout with ample whitespace for readability.
- Smooth, subtle animations and transitions to enhance user experience. Should remain professional, never distracting.
